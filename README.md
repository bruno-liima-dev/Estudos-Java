# Estudos Java

Repositório com projetos práticos desenvolvidos durante minha jornada de aprendizado em Java. Cada projeto foi construído do zero com foco em fixar conceitos reais de programação.

## Projetos

### 🔐 clienteSenha

Sistema de validação de senha com tentativas ilimitadas até o acerto.

**Conceitos praticados:**
- Laço `while` para repetição condicional
- Estrutura `if/else` para validação
- `Scanner` para entrada do usuário
- Operador `!=` para comparação

---

### 📋 cadastroPecas

Sistema de cadastro de peças automotivas utilizando arrays e entrada de dados pelo usuário.

**Conceitos praticados:**
- `Arrays` de String
- Laço `for` para percorrer e preencher o array
- `Scanner` para entrada do usuário
- Indexação base zero

---

### 🔢 tabuada

Gerador de tabuada com validação de entrada do usuário.

**Conceitos praticados:**
- Laço `while` para validação de intervalo (1 a 10)
- Laço `for` para geração da tabuada
- `Scanner` para entrada do usuário
- Operadores aritméticos e lógicos

---

### 📦 AutoStock

Sistema simples de estoque desenvolvido para praticar armazenamento de dados utilizando arrays.

**Conceitos praticados:**
- `Arrays` para armazenamento de dados
- `Scanner` para entrada do usuário
- Cadastro de informações
- Controle de posições no array

---

### 🔧 MenuOficina

Projeto de menu interativo para simular opções de uma oficina automotiva.

**Conceitos praticados:**
- Estrutura `switch`
- `Scanner` para entrada do usuário
- Estruturas condicionais
- Organização de menus

---

### 🔧 MenuOficinaV2

Evolução do projeto MenuOficina com melhorias na estrutura e controle de fluxo.

**Conceitos praticados:**
- Estrutura `switch`
- Laço `while`
- Menus interativos
- Controle de fluxo do programa
- Organização do código

---

### 📋 MiniEstoque

Sistema de estoque com cadastro, listagem e busca de peças automotivas.

**Conceitos praticados:**
- `Arrays` de String
- Laço `for`
- Estrutura `switch`
- `Scanner` para entrada do usuário
- Busca de dados em arrays
- Variáveis booleanas (`boolean`)
- Controle de posição no array
